# Button Stories

Let's define a story for our `Button` component:

<Story
  name="basic"
  args={{
    label: 'Submit'
  }}>
  {Template.bind({})}
</Story>

We can drop it in a `Canvas` to get a code snippet:

<Canvas>
  <Story 
    name="different title"
    args={{
      title: 'Another button',
      description: 'button 2',
    }}>
    {Template.bind({})}
  </Story>
</Canvas>
